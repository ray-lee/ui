Welcome to the CollectionSpace Test tenant, which has been configured to support a test collections. Additions to the core CollectionSpace functionality for this test tenant include support for scientific taxonomies, scientific determination, and more.

Log in to the demo using the email admin@testsci.collectionspace.org and password 'Adminstrator' for full access to the system; to view in read-only mode, use the email reader@testsci.collectionspace.org with password 'reader'.
